<h1 align="center">CLI SiteReconstructor</h1>

Основная команда:

```bash
npm run reconstruct -- --input <capture-folder>
```

## Параметры

- `--input <dir>`
- `--output <dir>`
- `--serve`
- `--assets`
- `--strict`
- `--port <number>`
