<h1 align="center">Assets Mode</h1>

Режим `--assets` добавляет в отчёт asset manifest и подготовленную структуру папок для дальнейшей работы с ресурсами.

## Запуск

```bash
npm run reconstruct -- --input <capture-folder> --assets
```
