<h1 align="center">Capture Format</h1>

`SiteReconstructor` reads a capture folder produced by `SiteSnapshotter` or `SiteCrawlerSnapshotter`.

## Core Files

- `page.json`
- `network.json`
- `assets.json`
- `endpoints.json`
- `schemas.json`
- `telemetry.json`
- `module_graph.json`
- `scenario.json`
- `security.json`
- `intelligence.json`
- `confidence.json`
- `evidence.json`
- `tech_tree.json`

It also understands crawler session files like `captures__<site>__...__page.json`.
