<h1 align="center">CLI</h1>

Main command:

```bash
npm run reconstruct -- --input <capture-folder>
```

## Flags

- `--input <dir>`
- `--output <dir>`
- `--serve`
- `--assets`
- `--strict`
- `--port <number>`

If `--output` is omitted, the report path is inferred automatically from the captured host.
