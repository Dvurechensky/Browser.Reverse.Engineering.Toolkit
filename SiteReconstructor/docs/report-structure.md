<h1 align="center">Report Structure</h1>

Typical report layout:

- `index.html`
- `assets/app.css`
- `assets/app.js`
- `data/*.json`
- `exports/*`

Key JSON files include `normalized.json`, `architecture.json`, `api_map.json`, `scenario_graph.json`, `module_graph.json`, `entity_map.json`, `storage_analysis.json`, and `security_report.json`.
