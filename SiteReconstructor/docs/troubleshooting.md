<h1 align="center">Troubleshooting</h1>

If the report looks empty or noisy:

- verify that the capture folder is complete
- repeat the capture in `intel` mode
- for crawler output, use the final session folder
- do not expect older captures to contain the newer IndexedDB intelligence layer
