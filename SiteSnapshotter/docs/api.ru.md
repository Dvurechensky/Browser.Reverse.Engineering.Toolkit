<h1 align="center">Публичный API SiteSnapshotter</h1>

Ключевые точки входа:

- `SiteSnapshotter.run(...)`
- `SiteSnapshotter.watch()`
- `SiteSnapshotter.stop()`
- `SiteSnapshotter.report()`
- `SiteSnapshotter.inspect(...)`
- `SiteSnapshotter.exportChunks(...)`
- `SiteSnapshotter.exportZip(...)`

Используй `run({ mode: "intel", download: false })` как основной режим для глубокой сессии.
