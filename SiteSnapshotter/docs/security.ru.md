<h1 align="center">Безопасность и ограничения</h1>

Инструмент:

- не обходит авторизацию
- не читает `HttpOnly` cookies
- не делает active scanning
- работает только с тем, что уже видно браузеру

Использовать его стоит только на тех сайтах и аккаунтах, где у тебя есть право на анализ.
