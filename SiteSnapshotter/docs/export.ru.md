<h1 align="center">Экспорт SiteSnapshotter</h1>

Рекомендуемый путь:

```javascript
await SiteSnapshotter.exportChunks(pack)
```

Если браузер режет множественные скачивания, используй:

```javascript
SiteSnapshotter.exportZip(pack)
```
